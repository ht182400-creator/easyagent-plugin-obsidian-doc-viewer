# Doc Project — 开发指南

## 环境要求

- Node.js >= 18
- pnpm >= 8

## 安装依赖

```bash
pnpm install
```

## 启动开发服务器

```bash
pnpm dev
```

访问 http://localhost:5184

## 运行测试

```bash
pnpm test
```

## 构建生产版本

```bash
pnpm build
```

## 代码规范

- TypeScript strict 模式
- 组件使用函数式组件 + Hooks
- 状态管理使用 Zustand
- 样式优先使用 Tailwind CSS

## 数据源说明

当前版本支持通过浏览器 File System Access API 读取本地目录：

1. 用户点击「打开目录」按钮
2. 调用 `window.showDirectoryPicker()` 选择文件夹
3. `src/services/fileLoader.ts` 递归扫描目录
4. 仅保留 `.md` 文件，构建目录树
5. 解析 `[[WikiLink]]` 并建立链接关系

### 文件系统 API 限制

- 必须在 `localhost` 或 `HTTPS` 环境下运行
- 需要用户主动点击触发（安全策略要求）
- 不支持在 iframe 或非安全上下文中使用

## 核心模块

| 模块 | 路径 | 职责 |
|------|------|------|
| 文件加载 | `src/services/fileLoader.ts` | 本地目录扫描、Markdown 读取 |
| WikiLink 解析 | `src/services/wikiLinkParser.ts` | 提取链接、解析相对路径 |
| 图谱构建 | `src/services/graphBuilder.ts` | 生成 D3 力导向图数据 |
| 搜索索引 | `src/services/searchIndex.ts` | FlexSearch 全文索引 |
| 全局状态 | `src/stores/docStore.ts` | 文档、图谱、搜索、设置状态 |
| 设置面板 | `src/components/SettingsModal.tsx` | 目录、图谱模式、搜索限制 |
