# Doc Project — 项目规划

## 项目目标

构建一个类似 Obsidian 的 Markdown 文档浏览器，支持：

1. 目录树浏览
2. 文档关系图谱
3. 节点点击预览
4. 全文搜索
5. 知识库向量索引 (远期)

## 范围

### MVP (最小可用产品)

- ✅ React + Vite + Tailwind 基础框架
- ✅ 左侧目录树 (资源管理器风格，仅显示 `.md`)
- ✅ 中间 D3 关系图谱 (全局 / 邻居 双模式)
- ✅ 右侧 Markdown 预览 (支持相对路径 WikiLink)
- ✅ 全文搜索 (标题 + 内容，高亮片段)
- ✅ 本地目录读取 (File System Access API)
- ✅ 设置面板

### 后续版本

- GitHub 仓库读取
- 向量语义搜索
- 插件化集成到 EasyAgent

## 关键指标

- 图谱渲染: ≤ 500 节点 60fps
- 搜索响应: ≤ 100ms (1000 文档)
- 首次加载: ≤ 2s
